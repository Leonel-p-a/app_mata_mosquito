# 🦟 App Mata Mosquito

Um jogo simples e divertido desenvolvido com HTML, CSS e JavaScript. O objetivo é eliminar os mosquitos que aparecem aleatoriamente na tela antes que o tempo acabe — se você perder três mosquitos, o jogo termina!

## 🚀 Funcionalidades

- Mosquitos aparecem em posições aleatórias na tela
- Vidas e tempo limite para completar o jogo
- Três níveis de dificuldade:
  - Médio
  - Difícil
  - Chuck Norris
- Versão responsiva: funciona em desktops e dispositivos móveis

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap
- JavaScript
